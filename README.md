# HTML 실습 과제

### 1. HTML(HyperText Markup Language)이란?
* 웹 페이지의 구조를 정의하는 언어
* 글의 제목, 문단, 이미지, 링크 같은 웹 페이지의 뼈대를 만드는 역할

### 2. 웹과 HTML의 관계
* 서버가 HTML 문서를 만들어서 브라우저에 전달
* 브라우저는 HTML을 읽고 화면의 기본 구조를 그림
* **프로세스**: 서버 → HTML 전달 → 브라우저 → 화면 구조 생성
### 3내부 실습내용
* 기본적인 태그들 적용해보고, li ul table같이 속성확인 img하고 <>안에 속성 입력해서 이미지 출력까지 확인
